# StringArt

## Name
String Art: Towards Computational Fabrication of String Images

## Description
This project has been developed for the PhD Exam Course "Python Fundamentals" at University of Milan. The code is based on the paper with the same title by Birsak et al. (DOI: [10.1111/cgf.13359](https://doi.org/10.1111/cgf.13359)).
